Frontend mini e-commerce project with product categories, wishlist (favorites), and shopping cart using `localStorage`.

## Features
- Product listing by categories (Jewelry, Food, Clothes, Electronics)
- Add/remove products to Cart
- Add/remove products to Favorites (Wishlist)
- Cart/Favorites counters in the header
- Data persistence using `localStorage`
- Separate pages: Home, Favorites, Cart, Login

## Tech Stack
- HTML
- CSS
- JavaScript (localStorage)

## Run Locally
Open `index.html` in your browser.
